# El Cartero Valiente
Descripcion: Un juego de plataformas 2D donde controlas a un cartero que debe cumplir su deber 
Objetivo: Recoger todas las cartas y paquetes esparcidos por el nivel antes de que acabe el tiempo
Desafio: Esquivar o saltar sobre los perros que patrullan las calles 
. Salta vallas y cercas para avanzar por los jardines
